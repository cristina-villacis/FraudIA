Exportar a Excel

Genera un archivo Excel multi-hoja optimizado para importar en Power BI con Dashboard Principal, Semáforo, Ramos, Proveedores, Temporal y Alertas.

Conexión Power BI

1. Haga clic en "Generar Exportación"

2. Descargue el archivo Excel

3. Power BI Desktop → Obtener datos → Excel

4. Seleccione el archivo descargado

5. Use semaforo_final para filtros de color

6. Use score_hibrido para distribuciones
