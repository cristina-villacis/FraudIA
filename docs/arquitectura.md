## Componentes Principales

### 1. Ingesta de Datos (`src/ingestion/`)
- Carga desde CSV, Excel o upload web
- ValidaciГіn de esquemas
- Limpieza automГЎtica (fechas, numГ©ricos, strings)
- Generador de datos sintГ©ticos

### 2. Feature Engineering (`src/features/`)
- Merge de tablas (pГіlizas, asegurados, vehГӯculos, proveedores, documentos)
- Variables temporales (borde de vigencia, reporte tardГӯo)
- Variables de frecuencia (asegurado, vehГӯculo, conductor, proveedor)
- Variables de montos (z-score por ramo, ratio vs suma asegurada)

### 3. Motor de Reglas (`src/rules/`)
- 12 seГұales de fraude con puntuaciГіn configurable
- 7 reglas crГӯticas (RF-01 a RF-07)
- Score normalizado 0-100
- SemГЎforo: Verde (0-40), Amarillo (41-75), Rojo (76-100)

### 4. Modelos ML (`src/models/`)
- **Supervisado**: Random Forest con balanceo de clases
- **AnomalГӯas**: Isolation Forest
- **HГӯbrido**: PonderaciГіn reglas (40%) + ML (35%) + anomalГӯas (25%)

### 5. NLP (`src/nlp/`)
- TF-IDF + Cosine Similarity para narrativas
- ExtracciГіn de entidades bГЎsica
- AnГЎlisis de frecuencia de palabras

### 6. Explicabilidad (`src/explainability/`)
- Explicaciones en lenguaje natural por caso
- Resumen ejecutivo agregado
- Recomendaciones por nivel de riesgo

### 7. Agente IA (`src/ai_agent/`)
- Procesamiento de consultas en lenguaje natural
- 12 tipos de consulta soportados
- Respuestas contextualizadas con datos

### 8. AplicaciГіn Web (`src/app/`)
- Dashboard interactivo con Plotly
- Upload de archivos
- Chat con agente IA
- ExportaciГіn Power BI

## Flujo de Datos

1. **Entrada**: Dataset vГӯa upload web o datos sintГ©ticos
2. **Limpieza**: ValidaciГіn, tipos, nulos
3. **Features**: 40+ variables derivadas
4. **Reglas**: Score basado en 12 seГұales
5. **ML**: Probabilidad de fraude + detecciГіn anomalГӯas
6. **Score Final**: HГӯbrido ponderado вҶ’ SemГЎforo
7. **Salida**: Dashboard, Agente IA, Power BI
